# Degree Audit Website
Website for tracking courses taken and degree progress.

### Current use cases
- UC0: Create User Account
- UC1: Log In
- UC4: Subnit Password Change Request

## Run

Clone the project
```
https://github.com/rnhkz/degree-audit-dev.git
```

Enter project directory
```
cd degree-audit-dev
```

Install
```
npm install
```

Start
```
npm start
```
